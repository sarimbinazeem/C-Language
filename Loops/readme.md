## üß© Topics Covered

### 1. `for` Loop
- Executes a block of code a known number of times
- Syntax: `for(initialization; condition; increment/decrement)`
- Ideal for counted iterations

### 2. `while` Loop
- Executes a block of code **while a condition is true**
- Condition is checked **before** each iteration
- Useful for unknown iteration counts that depend on runtime conditions

### 3. `do-while` Loop
- Similar to `while`, but condition is checked **after** executing the loop body
- Ensures the loop runs at least once

### 4. Nested Loops
- Loops inside loops
- Used for multidimensional data, patterns, or complex iterations

### 5. Loop Control Statements
- **`break`**: Exits the loop immediately
- **`continue`**: Skips the current iteration and continues with the next
- **`goto`**: Jumps to a labeled statement (use sparingly)
